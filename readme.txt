== Description ==

This plugin enables you to offer pro rata discounts on membership products within woocommerce
by creating a set of discount codes that are automatically associated to the products
of your selected caterories.

This plugin requires Woocommerce to be installed.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to settings > membership discounts to edit the plugin settings.
1. If you feel like it --donate some monies to https://www.paypal.me/martingreenwood.
